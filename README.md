# 智慧路灯管理平台

基于 Spring Boot 的智慧路灯后端服务，提供设备管理、光照监测、自动/手动控制、告警管理和实时推送功能。

## 通信通道说明

| 通道 | 协议        | 方向      | 触发方   | 用途                    |
|----|-----------|---------|-------|-----------------------|
| ①  | HTTP      | 前端 → 后端 | 前端    | 用户操作：CRUD、查询、手动控制、登录等 |
| ②  | WebSocket | 后端 → 前端 | 硬件/后端 | 实时推送：光照数据、状态变更、告警通知   |
| ③  | JDBC      | 后端 → DB | 后端    | 数据持久化                 |
| ④  | HTTP      | 硬件 → 后端 | 硬件    | 光照上报、心跳上报、开关状态回传、告警触发 |

## 数据流概览

### 光照监测 + 自动控制（事件驱动）

```
硬件传感器 ──→ POST /light-readings ──→ 保存光照记录
                                          │
                                          ├── 推送 WebSocket: LIGHT_REPORTED
                                          │
                                          └── 阈值判定
                                              ├── 光照 < 开灯阈值 → 自动开灯 (AUTO_ON)
                                              └── 光照 > 关灯阈值 → 自动关灯 (AUTO_OFF)
                                                  │
                                                  └── 推送 WebSocket: DEVICE_STATUS_CHANGED
```

### 心跳 + 离线检测

```
硬件 ──→ POST /devices/heartbeat ──→ 更新 lastHeartbeatTime
        (定期发送)                      设置 onlineStatus = ONLINE
                                         │
                                         └── 刚恢复上线? → 推送 WebSocket: DEVICE_ONLINE_STATUS_CHANGED

HeartbeatCheckTask (@Scheduled 30s) ──→ 扫描 ONLINE 设备
                                          │
                                          └── lastHeartbeat + timeout < now?
                                              │
                                              ├── 标记 OFFLINE
                                              ├── 创建 OFFICE 告警 → 推送 ALARM_CREATED
                                              └── 推送 DEVICE_ONLINE_STATUS_CHANGED
```

### 手动控制 + 状态回传

```
前端 ──→ HTTP 下发开关指令 ──→ 后端响应"指令已下发"
         (请求-响应)             │
                                ▼
                          硬件执行开关
                                │
硬件 ──→ POST /devices/status-callback ──→ 更新设备状态
         (回传最终状态)                     记录控制日志
                                            │
                                            └── 推送 WebSocket: DEVICE_STATUS_CHANGED
```

## 快速开始

```bash
# 1. 初始化数据库（PostgreSQL + pgvector）
psql -h localhost -U postgres -d smart-street-light -f sql/schema.sql

# 2. 创建敏感配置文件
cat > src/main/resources/application-secret.yml << EOF
spring:
  datasource:
    password: your_password
jwt:
  secret-key: your_secret_key
EOF

# 3. 启动应用
mvn spring-boot:run

# 4. 前端 WebSocket 连接
# ws://localhost:8080/ws?token=<jwt_token>
```
